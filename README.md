
## Installation

```bash
$ npm install
```

## Running the app

```bash
$ docker compose build
$ docker compose up -d
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
